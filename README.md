# network-diagnostic-toolkit
Python-based network diagnostic toolkit featuring ping, DNS lookup, TCP port checking, logging, and multithreading.

## Features

- Host Reachability Testing (Ping)
- DNS Resolution
- TCP Port Connectivity Checks
- Diagnostic Logging
- Multithreaded Operations

## Technologies Used

- Python
- Tkinter
- Socket Programming
- Subprocess
- Threading

## Skills Demonstrated

- Network Troubleshooting
- GUI Development
- File Handling
- Error Handling
- Multithreading
